# docker-laravel-alpine

This docker image contains anything needed to run a Laravel web application.

### Versions

Versioning follows classic SEMVER. Anything under the `2` tag (`2`, `2.1`, `2.1.2`, etc) will have PHP 7.3

| Tag | PHP Version |
|-----|-------------|
| `2`   | `7.3.x`       |
| `1`   | `7.2.x`       |
